## Data Flow Diagrams

### Message Analysis Flow

```
User Input
    в”Ӯ
    в–ј
POST /analyze-message
    в”Ӯ
    в”ңв”ҖвҶ’ Validate Input (Pydantic)
    в”Ӯ
    в”ңв”ҖвҶ’ MessageAgent.analyze()
    в”Ӯ   в”ңв”ҖвҶ’ Check Phishing Patterns
    в”Ӯ   в”ңв”ҖвҶ’ Check Scam Patterns
    в”Ӯ   в”ңв”ҖвҶ’ Check URLs
    в”Ӯ   в”ңв”ҖвҶ’ Check Sensitive Data
    в”Ӯ   в””в”ҖвҶ’ Return (category, risk_score, reasons)
    в”Ӯ
    в”ңв”ҖвҶ’ RiskEngine.calculate_severity()
    в”Ӯ   в””в”ҖвҶ’ Map score to CRITICAL|HIGH|MEDIUM|LOW
    в”Ӯ
    в”ңв”ҖвҶ’ RiskEngine.get_advice()
    в”Ӯ   в””в”ҖвҶ’ Generate category-specific advice
    в”Ӯ
    в”ңв”ҖвҶ’ PrivacyScrubber.scrub_sensitive_data()
    в”Ӯ   в””в”ҖвҶ’ Clean reasons (remove emails, phones, etc.)
    в”Ӯ
    в”ңв”ҖвҶ’ Construct RiskAnalysisResponse
    в”Ӯ
    в””в”ҖвҶ’ Return JSON Response
```

### URL Analysis Flow

```

    в”Ӯ   в”Ӯ   в””в”ҖвҶ’ Encoding analysis
```
